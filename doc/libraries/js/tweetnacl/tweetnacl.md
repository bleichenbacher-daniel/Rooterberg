# tweetnacl

**Language:** javascript\
**Url:**
[https://github.com/dchest/tweetnacl-js](https://github.com/dchest/tweetnacl-js)

The library implements primitives for the NACL protocol. As a consequence,
algorithms like xsalsa20-poly1305 are authenticated, but do not accept AAD.
