# k256

**Language:** rust\
**Url:** [https://docs.rs/k256/latest/k256/](https://docs.rs/k256/latest/k256/)\
**Tested version:** 0.13.4

This library generates normalized ECDSA signatures. I.e., the s value in the
signature satisfies 0 < s < n/2. Signature verification requires this format.
