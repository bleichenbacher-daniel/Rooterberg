# xtea

**Language:** rust\
**Url:** [https://docs.rs/xtea/latest/xtea/](https://docs.rs/xtea/latest/xtea/)\
**Tested version:** 0.1.0

XTEA does not define the endianness of its input. This crate uses little endian
encoding for keys, plaintext and ciphertext.
